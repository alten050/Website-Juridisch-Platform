# Jouw Fiscaal Recht - Multisite Website

Een professionele website voor een fiscaal juridisch bijstandsplatform, gebouwd met HTML, CSS (Tailwind), en JavaScript voor animaties.

## Functies
- Professioneel design met custom kleuren en typografie
- Responsief design (mobile-first)
- Animaties met CSS transforms en opacity
- Meerdere pagina's: Home, Over Ons, Diensten, Abonnementen, Contact, Login
- Veilige inlogopties met DigiD en e-Herkenning
- Abonnementen structuur gebaseerd op de beschrijving

## Pagina's
- `index.html`: Homepage met hero, voordelen, diensten overzicht, testimonials
- `over-ons.html`: Informatie over het bedrijf en team
- `diensten.html`: Gedetailleerde beschrijving van diensten
- `abonnementen.html`: Overzicht van abonnementen met prijzen
- `contact.html`: Contactformulier en informatie
- `login.html`: Inlogpagina met authenticatie opties

## Lokale Server
Om de website lokaal te bekijken:
```bash
python3 -m http.server 8000
```
Open http://localhost:8000 in uw browser.

## Deployment voor Multisite
Voor verschillende domeinen (JouwFiscaalrecht.nl, UwFiscaalrecht.nl, etc.), upload de bestanden naar de respectievelijke hosting accounts. Pas de navigatie aan indien nodig voor branding per domein.

## Technologieën
- HTML5
- Tailwind CSS (CDN)
- JavaScript voor animaties
- Google Fonts (Playfair Display, Inter)
